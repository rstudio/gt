# Location helper for targeting the source notes

`cells_source_notes()` is used to target all source notes in the footer
section of the table. This is useful for adding custom styles to the
source notes with
[`tab_style()`](https://gt.rstudio.com/reference/tab_style.md) (using
the `locations` argument). The 'source_notes' location is generated by
[`tab_source_note()`](https://gt.rstudio.com/reference/tab_source_note.md).

## Usage

``` r
cells_source_notes()
```

## Value

A list object with the classes `cells_source_notes` and
`location_cells`.

## Examples

Let's use a subset of the
[`gtcars`](https://gt.rstudio.com/reference/gtcars.md) dataset to create
a **gt** table. Add a source note (with
[`tab_source_note()`](https://gt.rstudio.com/reference/tab_source_note.md))
and style the source notes section inside
[`tab_style()`](https://gt.rstudio.com/reference/tab_style.md) with
`locations = cells_source_notes()`.

    gtcars |>
      dplyr::select(mfr, model, msrp) |>
      dplyr::slice(1:5) |>
      gt() |>
      tab_source_note(source_note = "From edmunds.com") |>
      tab_style(
        style = cell_text(
          color = "#A9A9A9",
          size = "small"
        ),
        locations = cells_source_notes()
      )
