# Location helper for targeting the footnotes

`cells_footnotes()` is used to target all footnotes in the footer
section of the table. This is useful for adding custom styles to the
footnotes with
[`tab_style()`](https://gt.rstudio.com/reference/tab_style.md) (using
the `locations` argument). The 'footnotes' location is generated by one
or more uses of
[`tab_footnote()`](https://gt.rstudio.com/reference/tab_footnote.md).
This location helper function cannot be used for the `locations`
argument of
[`tab_footnote()`](https://gt.rstudio.com/reference/tab_footnote.md) and
doing so will result in a warning (with no change made to the table).

## Usage

``` r
cells_footnotes()
```

## Value

A list object with the classes `cells_footnotes` and `location_cells`.

## Examples

Using a subset of the [`sza`](https://gt.rstudio.com/reference/sza.md)
dataset, let's create a **gt** table. We'd like to color the `sza`
column so that's done with the
[`data_color()`](https://gt.rstudio.com/reference/data_color.md)
function. We can add a footnote with
[`tab_footnote()`](https://gt.rstudio.com/reference/tab_footnote.md) and
we can also style the footnotes section. The styling is done with
[`tab_style()`](https://gt.rstudio.com/reference/tab_style.md) and
`locations = cells_footnotes()`.

    sza |>
      dplyr::filter(
        latitude == 20 &
          month == "jan" &
          !is.na(sza)
      ) |>
      dplyr::select(-latitude, -month) |>
      gt() |>
      data_color(
        columns = sza,
        palette = c("white", "yellow", "navyblue"),
        domain = c(0, 90)
      ) |>
      tab_footnote(
        footnote = "Color indicates height of sun.",
        locations = cells_column_labels(columns = sza)
      ) |>
      tab_options(table.width = px(320)) |>
      tab_style(
        style = list(
          cell_text(size = "smaller"),
          cell_fill(color = "gray90")
          ),
        locations = cells_footnotes()
      )
